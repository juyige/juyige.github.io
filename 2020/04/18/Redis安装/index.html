Redis命令行

  • 关闭redis
    • redis-cli -a password shotdown
    • ./redis_init_script stop
  • 进入redis客户端
    redis-cli
  • 输入密码
    auth pwd
  • 设置缓存
    set key value
  • 获取缓存
    get key
  • 删除缓存
    del key
  • 查看是否存活
    redis-cli -a password ping
